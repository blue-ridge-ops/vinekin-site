# VineKin Website

Marketing landing page for VineKin — a gamified Bible learning app for busy parents.

## Tech Stack
- Static HTML/CSS single-page site
- Firebase Hosting + Firestore (waitlist signups)
- Firebase App Check with reCAPTCHA Enterprise
- Google Fonts (Fraunces + DM Sans)

## Key Files
- `index.html` — entire site (HTML, CSS, JS all inline)
- `sitemap.xml` — single-page sitemap
- `robots.txt` — crawler directives
- `404.html` — branded error page
- `images/VineKinLogo.png` — primary logo (needs optimization)
